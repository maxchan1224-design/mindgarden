# MindGarden v0.2 — Awareness Companion 重新設計

呢份文件回應你嘅 redesign prompt,同時記錄今次 code 改動嘅理由。

## 1. 新定位

**"A lifelong awareness companion." — 一個陪你一世、幫你留意自己嘅安靜地方。**

唔係日記(日記係關於記錄),唔係 chatbot(chatbot 係關於回應),唔係治療師,唔係 productivity tool。核心動詞由「記錄」變成「留意」。

## 2. 核心哲學

一句講晒:**Notice, don't fix.**

- 覺察本身就係全部。唔需要進步、唔需要正面、唔需要每日快樂。
- 冬天都係季節。低潮唔係要修正嘅 bug。
- 愈少壓力,愈能長期維持。就算只揀咗一個季節,今晚都算完成。
- AI 嘅角色唔係傾偈,係「留意你自己留意唔到嘅嘢」。

## 3. USP

其他 AI journal(Rosebud、Reflectly、心光)嘅 AI 係**回應者** — 你寫,佢覆。
MindGarden 嘅 AI 係**觀察者** — 佢幫你留意 pattern:「呢兩星期你提咗返工八次,六次帶焦慮」「你呢排冇再提去旅行」。呢個係 awareness,唔係 chat。加上粵語原生、香港 crisis 資源、離線本機儲存,呢個組合冇人做緊。

## 4-6. User journey / 架構 / Onboarding

**開場唔再問「今日心情如何?」**(太普通,而且預設咗你要匯報情緒)。改問:

> Welcome back. Before we begin… **今晚你需要啲咩?**

四張大卡:🌿 我想慢落嚟 / 💭 我想清一清個腦 / ❤️ 我想有人聽下 / 🧭 我想諗清楚一件事。每張卡直接帶你去啱嘅練習 + 啱嘅對話風格,唔使用戶自己諗「我應該用邊個功能」。

導航剩返四個:**今晚(Reflect)/ 隨手記(Capture)/ 收藏庫(Library)/ 花園(Garden)**。冇 Insights,冇 Statistics,冇 dashboard — 一開始太重。設定收埋喺右上角。

## 7. AI 互動模型:風格取代人物

「阿晴/曉嵐/小澄」呢類 persona 嘅問題:佢哋係扮出嚟嘅人,關係愈深愈假。改成六種**傾偈方式**(唔係人物):

🌿 安靜陪伴 / 🪞 反思 / 🧠 理性思考 / 🌊 深度對話 / ⚡ 腦震盪 / ☀️ 鼓勵

對話中途隨時切換(對話框上面有個掣)。原有三把聲留低做「聲線」設定,只影響 TTS。

## 8. 每晚體驗

2–10 分鐘:揀 need → 做一個練習 → AI 用啱嘅風格接住 → 完。門檻降到最低:**只揀一個季節(🌱春 ☀️夏 🍂秋 ❄️冬)都算完成今晚嘅 check-in** — 季節取代咗「開心/唔開心」嘅二元框架,因為人唔係每日都要快樂,冬天都係成長。

**Silence**:每星期日,app 會話「今日唔使寫」,淨係陪你坐一分鐘(60 秒安靜計時)。一個會叫你唔好用佢嘅 app,先係一個你信得過嘅 app。

## 9-11. 長期旅程 / 記憶 / 成長系統

- **🌱 Seeds**:隨手記入面,一句重要嘅說話可以標做種子。佢喺花園跟時間生長:🌱(未夠七日)→ 🌿(生緊根)→ 🌳(已經係你一部分)。
- **花園分類**:記錄自動歸入 🌸感恩 🌿學習 🌻工作 🌳關係 🌹回憶 🍃放低 🌼夢想(本地關鍵詞分類,離線即時,唔使 AI)。唔係 folder,係一眼睇到你嘅人生而家開緊咩花。
- **⏳ 時間囊**:寫俾一年/三年/五年後嘅自己,封好之後時間未到打唔開(收藏庫會顯示鎖住嘅卡)。
- **👁 覺察(Notice)**:花園入面一個掣,AI 睇你最近十四日記錄,講返最多三個 pattern — 鐵律係只准陳述有記錄支持嘅觀察,唔准推測、唔准診斷、唔准建議。

## 12. Retention loop

唔靠 streak(植物用累積活躍日數,永遠唔會枯萎)。靠三樣:(1) 晚間儀式嘅低門檻(一個季節都算完成);(2) 種子同植物嘅緩慢生長 — 唯一睇到佢哋變化嘅方法係一路返嚟;(3) 時間囊 — 你自己埋低咗嘢喺度,你會返嚟開。

## 13-14. Monetisation / Premium

Free:每晚儀式、隨手記、花園、季節、Silence — 核心覺察永遠免費(呢個係信任嘅基礎)。
Premium 可以係:自然雲端 TTS 聲、加密雲端同步/跨裝置、每月/每季 AI 回顧(人生章節)、Emotional Map、無限時間囊、匯出精美年冊。原則:**收費嘅係深度同便利,唔係基本覺察。**

## 15. 移除咗嘅嘢同原因

- **Persona 人物選擇做核心** → 假關係。改做風格。
- **首頁功能列表**(情緒簽到/自我對話/身體覺察/小確幸四張卡)→ 呢啲係功能名,唔係需要。改做「今晚你需要啲咩」。
- **起伏 tab** → dashboard 太重,搬入花園做一個細掣(「睇下呢排嘅起伏」),想睇先睇。
- **「日記」呢個框架** → 改叫收藏庫(人生片段),減低「要寫嘢」嘅壓力。

## 16. 同競品嘅分別

- **Day One / Apple Journal**:記錄工具,冇覺察層,冇 AI 觀察。
- **Reflectly / Stoic**:guided prompt + mood tracking,但框架係「情緒好壞」;MindGarden 用季節,唔逼你快樂。
- **Rosebud**:最近似,但佢係英文、治療取向、AI 做回應者;MindGarden 係粵語原生、AI 做觀察者、離線本機。
- **心光**:AI 朋友主動回應 + 事件分類 — 佢賣「有人陪」;MindGarden 賣「見到自己」。分類係手段唔係賣點,花園先係。
